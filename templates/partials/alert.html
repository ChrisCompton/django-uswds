The alert can be customized by passing in the following variables: - heading: The heading of the alert. - message: The message to display in the alert. - style: The style of the alert. Options are 'info', 'warning', 'error', or 'success'. The default is 'info'. Example usage: {% include 'partials/alert.html' with heading='This is an alert' message='This is a message' style='info' %} For more information: https://designsystem.digital.gov/components/alert/ {% endcomment %}
